# SwiperIOS

## 一 说明

* 说明：基于APIOpen接口制作仿抖音IOS版
* 接口地址：https://api.apiopen.top/
* 实现功能：放抖音/某红书
* 功能模块：首页、图集、发布、消息、我

## 二 开发环境

* 系统：MacOS Sequoia 15.7.3
* IDE：Xcode(26.2)
* Expo SDK：54
* Node: 22.12.0
* Npm: 11.6.4
* AI助手：Trae
